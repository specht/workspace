## Summe mehrerer Zahlen berechnen

Implementiere eine Funktion, die die Summe aller Zahlen in einem Array berechnet.

<hr>

**Beispiel:**

<table>
<tr>
<td rowspan="1">Eingabe:</td>
<td>nums</td>
<td>=</td>
<td>[2, 7, 11, 15]</td>
</tr>
<tr>
<td>Ausgabe:</td>
<td colspan="3">35</td>
</tr>
</table>

Erklärung: Die Summe aller Zahlen im Array beträgt 2 + 7 + 11 + 15 = 35.

<hr>

*Achtung:* Falls das Array leer ist, soll die Funktion 0 zurückgeben.
