# shower.js für die Schule

In diesem Repository befindet sich ein angepasstes shower.js-Template, das für den Einsatz in der Schule geeignet ist.

Das originale shower.js-Projekt befindet sich hier: [https://github.com/shower/shower](https://github.com/shower/shower).

## Wie fange ich an?

Lade das Projekt herunter, indem du auf den blauen Button »Code« weiter oben drückst und den ganzen Ordner als zip-Datei herunterlädst. Speichere die Datei auf deinem H-Laufwerk, entpacke sie und öffne die Datei `index.html`. Dann kannst du dir die Präsentation im Browser anschauen.

## Wie kann ich die Präsentation verändern?

Um die Präsentation zu verändern, brauchst du einen Text Editor, mit dem man Textdateien bearbeiten kann.

Öffne die HTML-Datei z. B. mit Visual Studio Code, das auf unseren Schulrechnern schon installiert ist. Wenn du eine Änderung vorgenommen hast, kannst du sie im Browser sehen, indem du die Seite mit `F5` neu lädst.

## Tipp & Tricks für Profis

### 🚀 Shortcuts

- `Alt+Tab`: zwischen Fenstern wechseln
- im Browser:
  - `F5`: Seite neu laden
  - `F11`: Vollbild
  - `Esc`: Folienübersicht
- in Visual Studio Code:
  - `Strg+S`: Datei speichern
  - `Strg+B`: Seitenleiste ein-/ausblenden

### 🧭 Präsentation automatisch neu laden

Wenn du nicht nach jeder Änderung deine Präsentation im Browser neu laden möchtest, gibt es einen einfachen Trick. Schließe alle Dateien in Visual Studio Code und öffne stattdessen das Verzeichnis mit »File« / »Open Folder«. Links siehst du jetzt alle Dateien, öffne hier deine Präsentation.

Starte dann einfach den »Live Server« in Visual Studio Code (rechts unten), indem du auf »Go Live« drückst. In der Adresszeile deines Browsers steht jetzt nicht mehr `file://…`, sondern `http://…`. Das Coole daran: Wenn du deine Präsentation speicherst (z. B. mit `Strg+S`), dann lädt sich deine Präsentation im Browser automatisch neu. 😎

### 📷 Bilder

Wenn du Bilder einbindest, musst du den vollständigen Pfad zu deiner Bilddatei angeben. Leider ist Windows so schlau, Dateinamenerweiterungen wie `.png` oder `.jpg` auszublenden, was dieses Vorhaben torpediert. Du kannst dieses Verhalten entweder in den Ordneroptionen unter Windows ausstellen (bei »Bekannte Dateinamenerweiterungen ausblenden«) oder du nutzt einfach die Dateiansicht in Visual Studio Code (die linke Leiste).
